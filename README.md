2.	יצירת מסכים
יש לעצב מסכים עבור כל הפעולות הנדרשות במערכת:
מנהל יכול לבצע פעולות על לקוחות, מוצרים ומבצעים:
•	הצגת בודד
•	הצגת הכל (שימוש ב DataGridView) + אפשרות סינון (לכל ישות הגדירו אפשרות סינון אחת לבחירתכן)
•	הוספה
•	עדכון
•	מחיקה
קופאי יכול לבנות הזמנה.
בהוספת מוצרים להזמנה יש לאפשר הקשת קוד מוצר או בחירה מתוך רשימת המוצרים.
כמו כן, יש להציג בכל זמן את מצב ההזמנה הנוכחי – רשימת המוצרים והמבצעים וסכום סופי לתשלום
בסיום ההזמנה מבצעים DoOrder.

כל הפעולות במסכים יזמנו פונקציות של BL.
